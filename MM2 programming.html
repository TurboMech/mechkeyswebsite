Settings

Ensure boot loader size is set to 4096 KB, WS2812 LEDs should be set to 14, backlight levels set to 3 - 5, and ensure there are no errors.

Flashing your layout

After you've download your completed .hex, flash the firmware to your keyboard using QMK Toolbox.  Flashing the .hex is the same for all keyboards: put the keyboard in dfu-programming mode (by pressing the reset button on the bottom of the PCB), select the .hex within QMK Toolbox, and press flash! See our Programming How-To Guide for more instructions.
